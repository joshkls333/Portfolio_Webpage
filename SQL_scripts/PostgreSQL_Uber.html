Multiple Joins of same table many times:

 

For each trip that began before 2016 local time, select the firstname, lastname of a client, column original_fare_local and both names of his signup city and city in which trip took place. Rename columns to signup_city_name and trip_city_name.

 

SELECT

  cl.firstname,

  cl.lastname,

  ft.original_fare_local,

  sc.city_name as signup_city_name,

  dc.city_name as trip_city_name

FROM dim_client as cl

JOIN fact_trip as ft

  ON ft.client_uuid = cl.user_uuid

JOIN dim_city as dc

  ON ft.city_id = dc.city_id

JOIN dim_city as sc

  ON cl.signup_city_id = sc.city_id

WHERE begintrip_timestamp_local < '2016-01-01'

 

Complicated Joins:

 

Select user_uuid and his channel (show as user_channel) together with the uuid and channel of person who invited him. Rename inviter's columns accordingly: inviter_uuid and inviter_channel.

SELECT

  dc.user_uuid,

  dc.channel as user_channel,

  ic.user_uuid as inviter_uuid,

  ic.channel as inviter_channel

FROM dim_client as dc

JOIN dim_client as ic

  ON dc.inviter_uuid = ic.user_uuid

 

What was the average ETA of all completed trips in the US East region? Show it as avg_eta. Express your answer in minutes, rounded to 2 decimal places.

 

SELECT

  round(avg(ft.eta / 60.0),2) AS avg_eta

FROM fact_trip as ft

JOIN dim_city as dc

  ON dc.city_id = ft.city_id

WHERE dc.region = 'US East'

  AND ft.is_completed IS TRUE

 

How many drivers that completed trips in New Delhi (city_id=197) between December 1, 2015 and December 31, 2015 (use column dropoff_timestamp_local), currently have the status 'Active'? Show it as no_drivers.

 

 

SELECT

  count(DISTINCT dd.driver_uuid) as no_drivers

FROM

  dim_driver as dd

JOIN fact_trip as ft

  ON ft.driver_uuid = dd.driver_uuid

WHERE dd.status = 'Active'

  AND ft.is_completed IS TRUE

  AND ft.dropoff_timestamp_local between '2015-12-01' and '2015-12-31'

  AND ft.city_id = 197

 

Join drivers with their trips. For each driver select driver_uuid and app_ratingfrom dim_driver. For each trip, select uuid and vehicle_view_name from fact_trip.

 

 

SELECT

  dd.driver_uuid,

  app_rating,

  uuid,

  vehicle_view_name

FROM dim_driver dd

LEFT JOIN fact_trip ft

       ON ft.driver_uuid = dd.driver_uuid

 

For each client, show their first and last name together with the duration of their last trip. Show only clients whose lastname starts with the letter 'D'.

If there is no last_trip_uuid, show the client's name anyway.

SELECT

  firstname,

  lastname,

  trip_duration_seconds

FROM dim_client as dc

JOIN analytics_client ac

  ON dc.user_uuid = ac.user_uuid

LEFT JOIN fact_trip as ft

  ON ac.last_trip_uuid = ft.uuid

WHERE lastname LIKE 'D%'

 

Let's add another table to make things more complicated. For each client, show their first and last name together with the length of their first surge trip and the city name where it took place. Select only clients who are not banned and if there is no first_surge_trip, show the client's name anyway.

SELECT

  firstname,

  lastname,

  trip_distance_miles,

  city_name

FROM  ANALYTICS_CLIENT as ac

JOIN dim_client as dc

  ON ac.user_uuid = dc.user_uuid

LEFT JOIN  FACT_TRIP as ft

  ON ft.uuid = ac.first_surge_trip_uuid

LEFT JOIN dim_city as ci

  ON ac.first_surge_trip_city_id = ci.city_id

WHERE dc.banned IS FALSE

 

How many active US drivers whose flow is P2P, do not have the same current city as their signup city?

 

SELECT

  COUNT(*)

FROM

  dim_driver as dd

JOIN dim_city as dc

       on dc.city_id = dd.flow_city_id

WHERE dd.status = 'Active'

AND dd.driver_flow = 'P2P'

AND dd.signup_city_id != dd.flow_city_id

AND dc.country_name = 'United States'

 

How many trips in Bangalore (city_id = 130) in December 2015, were completed by drivers that have app_rating higher than 4? Rename the result to bangalore_2015_tripsand use column dropoff_timestamp_local to count the trips.

 

SELECT

  COUNT(t.uuid) as bangalore_2015_trips

FROM fact_trip as t

LEFT JOIN dim_driver as d

  on d.driver_uuid = t.driver_uuid

LEFT JOIN dim_city as c

  on c.city_id = t.city_id

WHERE d.app_rating > 4

AND c.city_id = 130

AND t.dropoff_timestamp_local between '2015-12-01' AND '2015-12-31'
